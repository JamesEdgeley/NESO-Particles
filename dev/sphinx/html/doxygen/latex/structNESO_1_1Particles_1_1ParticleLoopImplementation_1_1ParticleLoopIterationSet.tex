\doxysubsection{Detailed Description}
For a set of cells containing particles create several sycl\+::nd\+\_\+range instances which cover the iteration space of all particles. This exists to create an iteration set over all particles which is blocked, to reduce the number of kernel launches, and reasonably robust to non-\/uniform.
