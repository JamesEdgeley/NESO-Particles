This function performs the all-\/to-\/all movement that occurs when a \mbox{\hyperlink{classNESO_1_1Particles_1_1ParticleGroup}{Particle\+Group}} contains particles with positions (far) outside the owned region of space. For example consider if each MPI rank creates N particles uniformly distributed over the entire simulation domain then the call to {\ttfamily hybrid\+\_\+move} would have a cost equal to the number of MPI ranks squared.

This function gives each particle a temporary position in the owned subdomain then moves the particles in a straight line to the original positions in the positions Particle\+Dat. It is assumed that the simulation domain is convex.

This function is used by adding particles with {\ttfamily \mbox{\hyperlink{classNESO_1_1Particles_1_1ParticleGroup_a6770d444576949f95e15c0a21c2b9580}{Particle\+Group.\+add\+\_\+particles\+\_\+local}}} on each rank then collectively calling this function.
