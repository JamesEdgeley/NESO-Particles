class NESO\+::\+Particles\+::\+Particle\+Loop$<$ KERNEL, ARGS $>$}

\mbox{\hyperlink{classNESO_1_1Particles_1_1ParticleLoop}{Particle\+Loop}} loop type. The particle loop applies the given kernel to all particles in a \mbox{\hyperlink{classNESO_1_1Particles_1_1ParticleGroup}{Particle\+Group}}. The kernel must be independent of the execution order (i.\+e. parallel and unsequenced in C++ terminology).
